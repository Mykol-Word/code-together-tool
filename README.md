# Code Together Embed

A tiny GitHub Pages site for embedding YouTube videos in Code Together.

## Usage

```
https://mykol-word.github.io/code-together-tool/embed/youtube/?videoId=ucRulNQsuYQ&volume=20&startTime=0
```

## Parameters

- `videoId`: YouTube video id
- `volume`: player volume from `0` to `100`, defaults to `20`
- `startTime`: start time in seconds, defaults to `0`

`videoID`, `v`, and `voluem` are also accepted.
