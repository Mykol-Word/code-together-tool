# Code Together Embed

A tiny GitHub Pages site for embedding YouTube videos in Code Together.

## Usage

```
https://code-together-tool/embed/youtube/?videoId=ucRulNQsuYQ&volume=60&startTime=12
```

## Parameters

- `videoId`: YouTube video id
- `volume`: player volume from `0` to `100`
- `startTime`: start time in seconds

`videoID`, `v`, and `voluem` are also accepted.
